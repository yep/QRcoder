# QRcoder - QR-Code Generator

Easily generate custom QR-Codes.

Allows for sharing (iOS) or saving (macOS) the created QR-Code. 

## Download

See [Apple AppStore](https://apps.apple.com/us/app/qrcoder-qr-code-encoder/id1545201375)
